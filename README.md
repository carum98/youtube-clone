**YouTube clone** build with Vue 3 + Typescript + Vite

## SetUp
This project use [YouTube API](https://developers.google.com/youtube/v3)

Create ``.env``:

```
VITE_YOUTUBE_URL=[GOOGLE_YOUTUBE_URL]
VITE_YOUTUBE_API_KEY=[YOUTUBE_API_KEY]
```
